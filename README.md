# Premium Auto Lounge

## Описание проекта

Интерактивный веб-сайт для премиального автосалона, позволяющий клиентам:
- Просматривать каталог премиальных автомобилей
- Использовать онлайн-конфигуратор для настройки автомобилей
- Оставлять заявки и отзывы
- Получать информацию о покупке и доставке

## Тема проекта
**Онлайн-платформа премиум-автосалона с конфигуратором автомобилей**

## Используемый фреймворк
**Bulma**
